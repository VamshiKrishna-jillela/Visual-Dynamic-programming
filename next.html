Dynamic Programing Visualizer.
Assume You are Mario.When You pass by the road You have two choices either you can take upper path or lower path. To Take the upper path you need to jump. As you are weak, You Can't take two consecutive upper paths.i.e.., When you took upper path, you are forced to take the lower path next time. Each Path contains two certain number of coins. You Can collect coins avaliable in the path you are travelling. You are given a 2D matrix. You'll receive a 2D array as input. Each column stores rewards on higher and lower platform. You have to return an array stating 1 if you want to jump to higher platform, 0 otherwise. However, after selecting 1 you have to select 0 the next time. Can you solve it ?
